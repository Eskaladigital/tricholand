# Tricholand Web

Web corporativa de Tricholand — vivero productor de Trichocereus y cactáceas columnares. Venta B2B para profesionales.

## Stack

- **Next.js 15** (App Router)
- **React 19**
- **TypeScript**
- **Tailwind CSS 4**

## Idiomas

7 idiomas: ES, EN, NL, FR, DE, IT, PT

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```
